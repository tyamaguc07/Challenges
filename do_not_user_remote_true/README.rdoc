== README

=== 準備
以下のコマンドを実行してください。

1. bundle
2. rake db:migrate
3. rake db:seed
4. rails server

起動したサーバに以下のURLでアクセスし、問題なく動いていることを確認して下さい。
http://localhost:3000/posts
(URLのホスト部は環境によって変わります）
